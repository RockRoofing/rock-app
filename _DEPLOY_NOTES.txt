CHANGE
------
Each tab now opens with the month filter pre-set to the most recent COMPLETE month
(the previous calendar month — the current month is partial), instead of "all
months".

Fallback logic (so it never lands on an empty view):
  • If that tab has data for last full month -> use it.
  • Else -> the newest month present in that tab's data.
  • Else (no dated data) -> all months.

Applies on first load and whenever you switch tabs (each tab picks its own most-
recent full month). You can still change the month or pick "All months" manually.
