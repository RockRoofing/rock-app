CHANGE
------
Downloaded form PDFs are now named:
  [Project Number] [Project Name] - [Form Name] [Date Submitted]
e.g.  J190-Russell-Hill-Water-Ingress-2026-07-15.pdf
(The project number + name come from the submission's project field, which already
holds "J190 — Russell Hill". Date is the submitted date, yyyy-mm-dd.)

• Separate downloads: each file named as above for its own form.
• Combined download: named after the EARLIEST-dated selected form, with "Combined"
  appended, e.g. J190-Russell-Hill-Water-Ingress-2026-07-15-Combined.pdf
• Single downloads (row / modal "Download PDF") use the same naming.

Note: spaces and special characters (incl. the — dash) are converted to hyphens so
the filename is safe on Windows/Mac. The order of the fields matches the spec.
