THE LAST STEP
-------------
Consent now works (Manual journals scope is granted ✓). But the token exchange
failed with: invalid_grant / "Invalid redirect_uri".

Cause: the callback built its redirect_uri from NEXT_PUBLIC_APP_URL, falling back
to an OLD domain (rockroofing-app.vercel.app). Xero requires the token-exchange
redirect_uri to EXACTLY match the one used at authorize (app.rockroofing.co.uk).
Mismatch -> invalid_grant.

FIX
---
The callback now derives redirect_uri from the actual request host
(app.rockroofing.co.uk) instead of the stale env var, so it always matches the
authorize step. No env var needed.
