CHANGE (your suggestion — better design)
----------------------------------------
Instead of a separate "upload all at once" box with type auto-detection, each of
the three boxes (Bills, Direct Wages, Sales Invoices) now accepts MULTIPLE files
of its own type. Select several 500-line batches in one box and they upload one
after another, each merging/deduping into the same store.

• No auto-detection needed — each box already knows its type (more reliable).
• Keeps each box's own export instructions and detailed result panel.
• Per-file result shown under the box (filename + its result), so you can see
  each batch's "new / untagged / archived" outcome.
• The separate all-at-once box + detection logic have been removed.

So: to load 3 years of Bills, export in ~8 batches and select them all together
in the Bills box — one click imports the lot.
