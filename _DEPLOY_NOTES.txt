CHANGES
-------
1) MISSING-DATES BANNER (like the Application Calendar)
   Projects with no application dates set now appear in an amber "⚠ Missing dates:"
   banner ABOVE the project dropdown — outside the table — listing each project as
   a clickable link (opens the project so dates can be set). They no longer appear
   as rows in the table.

2) TABLE TITLE
   The table is now titled "Upcoming Applications".

3) DROPDOWN LABEL
   The project selector label now reads "Select Project — Upcoming Applications".

4) DATE NEEDED
   The table's date column is now headed "Application date needed" and shows the
   date each application is required (computed from the project's application day /
   overrides), colour-coded: grey not-yet-due / orange within 3 days / green due
   today / red overdue.

Table still shows only projects whose next application falls within 31 days, max
one row per project, soonest first.
