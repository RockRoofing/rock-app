Deploy target: RockRoofing/rock-app  branch main

Deploy over the top.

WHY
---
The "In Xero" figures come from the P&L benchmark, which was only pulled by the
nightly sync — so until it ran, Xero figures showed "—" (your _diag had
benchmark:null). This adds an on-demand refresh.

NEW: "↻ Sync Xero figures" button (top-right of the Bookkeeping page)
--------------------------------------------------------------------
• Pulls ONLY the Profit & Loss benchmark for the last 6 months — cheap
  (~6 Xero API calls, one per month), so it's safe to press whenever you like.
  It does NOT run the heavy full deep-sync (no per-project rotation / bill sweep).
• Refreshes the page data on completion, so the Xero column + graph benchmark
  populate immediately. Shows "Synced N months" / an error message.
• Merges with any existing benchmark months (won't lose older months).
• Accounts / Management / Admin only.

ALTERNATIVE (no button needed): you can still force the FULL nightly sync by
visiting /api/cron/deep-sync while logged in — but that's heavier and slower.
The button is the lightweight option for just the Xero figures.

NOTE: needs Xero connected with a valid refresh token. If it says "reconnect
Xero", the token has expired and needs re-authorising in the Xero settings.
