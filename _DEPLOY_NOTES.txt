PLANNING — Send WLA week model + historic View button

  pages/operations/project-management/planning.js

  SEND WLA POP-OUT — new week model:
   - "How far ahead" = 1 / 2 / 3 / 4 weeks, where WEEK 1 = NEXT week (the Monday after the current
     week; if today is Monday, still next Monday). Weeks 2–4 follow. Default = 1 week.
   - New "Include previous week (download only)" checkbox — OFF by default; when off the previous-week
     table is hidden. When ticked it shows the previous week but it is NEVER emailed (download only).
   - The current week (the one we're in) is no longer part of Send.
   - Emails cover the forward weeks only; operative de-select applies to those. Download PDF spans the
     selected forward weeks (plus previous week if ticked).

  NEW "View Weekly Labour Allocations" button (next to Send, ghost style):
   - Opens a historic/upcoming viewer. Pick any Week Commencing date (snaps to that week's Monday),
     see the per-operative grid + colour key. DOWNLOAD PDF ONLY — no send.

Deploy over previous planning.js. No API change (uses existing planning-week / -pdf).
