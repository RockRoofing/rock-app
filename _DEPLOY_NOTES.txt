1) MATERIALS GROUPED BY SUPPLIER (with attachable docs)
   • "Add all lines" from a PO now inserts a SUPPLIER HEADING row above that PO's
     line items in Materials on Site. Two POs from the same supplier = the supplier
     name appears twice (one heading per PO added), as requested.
   • Each supplier/group heading has "+ Attach doc" — documents attach to that
     group and will be appended to the application (Phase 4). Listed with links,
     removable. "Remove group" removes the heading and its lines.
   • Group headings don't total (only the item lines do). Adding a single line
     (the per-line "Add") still adds an ungrouped line as before.

2) NEW APPLICATION STARTS FROM THE PREVIOUS ONE
   Creating a new application now CLONES the latest application exactly:
   • Contract Works % complete carried across (matched by line).
   • Materials on Site copied in full — supplier groups, lines, mark-ups, % claimed
     AND their attachments (group ids are re-linked correctly).
   • Per-variation data (% complete + attachments) copied.
   • MCD % / retention % carried from the previous app.
   Only the identity changes automatically: application number (seq +1), month
   (auto-advances to the month after the last app), and App/Val/Payment/Final
   dates (recomputed from Project Details for the new month).
