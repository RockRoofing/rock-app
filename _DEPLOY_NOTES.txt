WIP — clear the project WIP margin override per section
======================================================
Deploy target: RockRoofing/rock-app  branch main
Build: Compiled successfully

CHANGE
------
Each project section on the WIP page can now clear its WIP margin OVERRIDE and revert
to the calculated (live) margin:
  • When a project has a manual margin override, a "Clear override" button appears next
    to the margin box. Clicking it removes the override so the project uses the live
    calculated margin ((invoiced − costs) / invoiced) again.
  • Under the margin it now shows whether the current margin is an "override
    (calculated: X%)" or the "calculated margin", so it's clear which is in effect.

API now returns marginIsOverride + calculatedMargin per project so the page can show
this and offer the clear button.
