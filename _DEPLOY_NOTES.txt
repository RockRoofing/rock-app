CHANGE
------
The [Sub-Contract Ref] merge field (used in the Suspension of Performance /
withdrawal email, and any template that uses it) now pulls the ORDER REFERENCE
from Project Details:
      order-ref override (Project Details) → IHM customerOrderRef → blank
This is the same value shown as "Order reference" in Edit Project Details, so
the two always match.

NOTE ON REFRESH
---------------
The invoice row reads this from the dashboard cache (4h). If you've just changed
the order reference in Project Details, it appears in the email after the next
dashboard sync/refresh (Project Details itself updates immediately).
