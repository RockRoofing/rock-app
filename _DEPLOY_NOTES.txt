Contracted Rates — drag reorder, headings, variations, colours
==============================================================
Deploy target: RockRoofing/rock-app  branch main
Build: ✓ Compiled successfully  ·  tab shows "· v4"
Deploy on top of the Contracted Rates work.

1) DRAG TO REORDER
------------------
Rows can now be dragged up/down (grab the ⋮⋮ handle at the left of each row) to
reorder within their section. The ⋯ menu's "Move up / Move down" still work too.

2) MOVE-UP BUG FIXED
--------------------
Adding a line to the bottom of the above-the-line section and not being able to
move it up is fixed. New lines now insert at the end of THEIR OWN section, and
reordering operates within the section (previously it compared array neighbours,
which could be a below-the-line row, so it refused). Verified with a test.

3) HEADINGS BOLD + UNDERLINED (toggleable)
------------------------------------------
Heading rows (a description with no rates/totals) now render BOLD + UNDERLINED.
Each heading's ⋯ menu has "Plain heading (no bold/underline)" to turn the styling
off for that heading (and back on again).

4) TURN A BELOW-THE-LINE ITEM INTO A VARIATION
----------------------------------------------
Below-the-line rows have a new ⋯ menu item "Turn into variation". It adds the item
to THIS project's variations (the Variation Tracker / project settings), mapping:
   materials total -> materials, labour total -> labour,
   remainder of the rate -> profit, description/code carried over,
   starting as NOT instructed. (New API action: to-variation.)

5) COLOURS + SEPARATORS
-----------------------
• Vertical separator lines between all columns.
• Materials rate + Materials total columns = LIGHT blue.
• Labour rate + Labour total columns = DARKER blue.
• Top summary cards recoloured to match (Materials light blue, Labour darker).
