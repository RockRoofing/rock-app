DEPLOY NOTES - Business Financials > Budgets rebuilt as an Overheads P&L grid
=============================================================================

WHAT IT NOW DOES (as specified)
-------------------------------
The Budgets page is now a mirror of the Overheads P&L for a financial year:

  - Rows: every account categorised OVERHEADS in Account Categorisation, shown as
    Code (far left) then Account name.
  - Columns: the 12 months of the financial year (1 Dec - 30 Nov).
  - Financial-year switcher (top right): "Dec 2024 - Nov 2025 (FY2025)" etc.
  - Per line, a FORECAST method selector: 1 = average of completed months this FY,
    2 = trailing 3 months, 3 = same month last year. Future months autopopulate
    from the chosen method.
  - Per month cell:
      * Completed month -> the ACTUAL overhead spend from Xero, coloured GREEN if
        at/under budget, RED if over budget.
      * Future month -> the FORECAST in light grey; type over it to override. When
        the month completes, the actual replaces it automatically.
  - Under each figure is a small BUDGET box (per month, individually editable). When
    you enter a budget and leave the box, it asks "Apply this budget to the
    following months in this year too?" - Yes fills it forward across the FY.
  - "Save" persists budgets, forecast methods and overrides.

DATA SOURCE
-----------
Actuals come from the P&L benchmark (xero:pl-benchmark), per account code per month.
sync-benchmark now pulls 13 months (was 6) so a full FY + same-month-last-year is
available. IMPORTANT: to populate history, click "Sync Xero figures" on a Business
Financials page. Older months accumulate as you sync over time.

New Redis keys (created automatically on first save):
  config:overhead-budgets            { code: { 'YYYY-MM': amount } }
  config:overhead-forecast-methods   { code: 1|2|3 }
  config:overhead-forecast-overrides { code: { 'YYYY-MM': amount } }
