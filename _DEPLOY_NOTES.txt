CHANGES (pages/project/[id].js — WIP tab manual adjustments)
------------------------------------------------------------
• ONE adjustment type only: "Cost adjustment", accepting POSITIVE or NEGATIVE
  amounts. Positive = cost incurred but not yet in Xero (increases WIP). Negative
  = cost you won't recover / WIP write-down (decreases WIP). The old "Invoice"
  type is removed — it never actually affected WIP and was redundant with the
  invoice sync. WIP = (post-val costs + net cost adjustment) / (1 − margin).
• Adjustments are now EDITABLE inline (Edit → change month/description/amount →
  Save), in addition to delete.
• The month selector DEFAULTS to the last full month (previous calendar month),
  still fully changeable.
• The Add button shows the month it will apply to by NAME, e.g. "+ Add to June
  2026" — so you set and confirm the month at the same time.
• Footer now shows a single "Net cost adjustment" for the month.

This file also carries the Group A Project Details changes (Income key/exact
values, WIP-tab filters, WIP Margin Override label, Invoiced Inc. Retention
rename, variations default). It is the latest full version of the file — use it.
