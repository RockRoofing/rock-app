WIP — clear a manual adjustment's margin to inherit the project margin
======================================================================
Deploy target: RockRoofing/rock-app  branch main
Build: Compiled successfully

CHANGE
------
You can now clear the margin on a manual WIP adjustment so it falls back to the
project's current margin (and tracks it if the project margin later changes):

• EXISTING adjustments now show an editable margin box with a "Reset" button. Blank it
  (or click Reset) and Save -> the adjustment inherits the project margin, shown as
  "(project X%)". Type a value + Save to set a specific margin.
• The ADD form's margin box has a "Reset" button too, to blank it back to the project
  margin before adding.
• A blank/cleared margin is stored as null and the WIP calc uses the project margin
  for that line.

API: added a PATCH handler to wip-adjustments to update an adjustment's margin
(margin=null clears it).
