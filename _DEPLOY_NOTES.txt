CHANGES (pages/project/[id].js — Costs tab)
-------------------------------------------
1. LABOUR / MATERIALS FILTER
   New dropdown: "Labour & Materials" / "Labour only" / "Materials only".
   (Clicking the budget rows still toggles it too — both drive the same filter.)

2. SEARCHABLE SUPPLIER FILTER
   The supplier dropdown is now a type-to-search combobox (same as Bookkeeping):
   click, type, list narrows; pick one; "All suppliers" clears.

3. ACCOUNT FILTER — FIXED + MULTI-SELECT BY CODE
   The old "All accounts" did nothing because it filtered on accountName, which
   is blank on most cost lines (they carry accountCode). It now:
     • lists account CODES actually present (shows "code — name" where a name
       exists),
     • is MULTI-SELECT via checkboxes in the dropdown,
     • has a search box to find a code/name,
     • filters the lines to the ticked codes.
   This is what you'll use to spot a wrongly-categorised high-value code driving
   the inflated labour/materials spend.

Note: the WIP tab's own (separate) filters are unchanged.
