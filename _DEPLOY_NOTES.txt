PROJECT REPORT — only include issues sent to the customer

Rule applied: an issue is included in the report's Issues table AND appended (full form) ONLY if it has
been sent to the customer OR marked as sent (sentToCustomer === true OR sentManually === true). Issues
marked "No — Do not send" (sendToCustomer === 'nosend') and any not-yet-sent issues are excluded.

  pages/operations/project-management/report.js   Issue auto-pull on project select now applies this filter.
  pages/api/project-report-pdf.js                 Enforces the same filter when appending full forms (so
                                                  even older saved reports won't append excluded issues).
  lib/projectReportPdf.js                         The PDF's issues table renders only the included issues,
                                                  so the table and appended forms always match.
