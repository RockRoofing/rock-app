FAVICON — Rock Roofing logo as the favicon on every page (Portal + Site App)

WHAT'S INCLUDED
  pages/_document.js  (NEW) — global document that adds the favicon / apple-touch-icon / manifest <link>
    tags to EVERY page (portal + Site App), so the Rock logo shows in the browser tab everywhere.
  public/favicon.ico            (16/32/48/64 multi-res)
  public/favicon-16x16.png
  public/favicon-32x32.png
  public/apple-touch-icon.png   (180x180 — iOS "Add to Home Screen" icon)
  public/favicon-192.png        (Android home-screen)
  public/favicon-512.png        (PWA/large)
  public/site.webmanifest       (names the app "Rock Roofing" + home-screen icons)

  All generated from public/rock-logo.jpg.

DEPLOY NOTE (important for the binary files)
  - Put the pages/_document.js file in pages/.
  - Put ALL the public/* files in the public/ folder at the repo root.
  - When uploading images via the GitHub web editor, use "Add file > Upload files" and drag the PNGs/ICO in
    (don't paste them as text — they're binary and will corrupt if pasted).
  - Browsers cache favicons aggressively; after deploy you may need a hard refresh (or check in a private
    window) to see the new tab icon.

No cron/env change.
