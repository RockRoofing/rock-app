HOW IT WORKS NOW
----------------
Anticipated VAT = Output VAT (on sales) minus Input VAT (on purchases), by month:
  Output VAT  = TotalTax on authorised/paid ACCREC invoices, minus ACCREC credit notes
  Input VAT   = TotalTax on authorised/paid ACCPAY bills, minus ACCPAY credit notes
  Net VAT     = output - input   (negative = REFUND due from HMRC; positive = payable)
Grouped by transaction month. The page shows: this month's position, period totals, a
net-VAT-by-month chart (green below zero = refund), and a month table with sales/purchases
net, output/input VAT, net VAT and the position.
