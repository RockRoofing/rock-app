Two small, self-contained fixes from the Financial App brief. The larger items
(retention Xero columns: Invoiced/Paid/VAT/balance; manual-vs-existing-project
entry with write-back; overdue-invoice list + automated email folded into the
master sync cron) are coming in the next batch.

1) APPLICATION CALENDAR — "Payment due" date off-screen
-------------------------------------------------------
The project popup showed Application / Valuation / Payment-due in a fixed 3-column
grid; on narrower screens the native date pickers pushed "Payment due" past the
modal edge. The grid is now responsive (auto-fit, wraps as needed) and the date
inputs can shrink, so all three dates stay on screen.

2) PROJECT DETAILS — retention comments box
-------------------------------------------
In Edit Project Details → Retention, under "Defects liability end date", there is
now a double-height "Retention comments" text box. It saves with the rest of the
project settings (persists to the project record; no data migration needed).
