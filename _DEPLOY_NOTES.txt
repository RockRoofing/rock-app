KEY CHANGES
-----------
• Applied for — new column. Manual/override for now; will auto-populate from the
  Application tab once that's built (editable in the Edit form).
• Invoiced → "Invoiced Net" — total invoiced EX-VAT (Xero for new projects, or
  the imported Xero CSV for existing).
• Balance → "Account Remaining" = Final Account − Invoiced Net.
• VAT = Final Account × the VAT-type rate (20%/5%); reverse charge / 0% = £0.
• VAT Type — reverse charge / 5% / 20% / zero-rated etc (from Xero; editable).
• Total Due = Final Account + VAT.
• Paid → "Total Paid" (inc VAT).
• Total Remaining (Check) = Total Due − Total Paid. When it hits £0 the whole
  ROW turns green — the signifier that retention is closed and can be moved off
  the list (switch the toggle to see closed ones).
• Status column removed.

RELEASES (1st / 2nd Value)
--------------------------
The separate "Rcvd" tick columns are gone. Each release Value cell is now:
  • ORANGE with "due" underneath when not yet received
  • GREEN with "received" underneath once marked received
Mark a release received in the row's Edit form. A colour key sits above the table.

TOGGLE (replaces the All/Pending/Overdue/Released tabs)
-------------------------------------------------------
A single toggle: "Outstanding" (default — only rows where Total Remaining ≠ £0)
and "All" (every project that has/had retention).

HEADER TOOLTIPS
---------------
Every column header has an ⓘ hover note explaining where the number comes from /
how it's calculated.
