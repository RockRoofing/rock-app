Applications — re-send warning, auto-CC sender, PDF spacing (no lines)
=====================================================================
Deploy target: RockRoofing/rock-app  branch main
Build: ✓ Compiled successfully  ·  Applications tab shows "· v23"

1) RE-SEND WARNING
   If an application has already been sent, pressing "Send email" in the send
   popout asks: "Are you sure you want to send this application to the customer
   again?" before sending.

2) AUTO-CC THE SENDING PORTAL USER
   The person sending is automatically copied in (their email is pre-ticked in CC,
   and always included on send even if unticked).

3) PDF — LINES REMOVED, MORE ROW SPACING
   All table gridlines (vertical + horizontal) have been removed. Instead there's
   more vertical space between each line item so every item reads clearly against
   its qty / rate / total. Applies to Summary, Payment Certificate, Contract Works,
   Variations and Materials on Site. The section-divider rules under headings and
   above the section totals are kept (those are structure, not table lines).
