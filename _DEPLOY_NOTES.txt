CHANGE
------
Clicking a project in the amber "⚠ Missing dates" banner now opens a date-entry
popup (the same recurring Application day / Valuation day / Payment day fields the
Application Calendar's banner opens) so you can set the dates right there. On Save
the dates are written to the project settings and the Upcoming Applications table +
banner refresh immediately — the project moves out of the banner and into the
table once its next application date is known.
