CHANGE
------
The sticky selection bar (the same one used for combining variations) now has
bulk formatting buttons: B (bold), U (underline), A (red). Tick any lines —
above AND/OR below the line — then click a format to apply it to all selected
lines at once. Toggle behaviour: if any selected line doesn't have the format it
turns ON for all; if they all already have it, it turns OFF for all.
Per-line formatting via the row ⋯ menu still works too.
