WIP nav — add missing separator lines around WIP / Commercial Scorecard
=======================================================================
Deploy target: RockRoofing/rock-app  branch main
Build: Compiled successfully

ISSUE
-----
When the WIP link was added to each page's inline nav, its trailing "|" separator was
missed, so:
  • On most pages there was no line between WIP and Commercial Scorecard.
  • On the Commercial Scorecard page there was no line between Application Calendar
    and WIP.

FIX
---
Added the missing "|" separators so the nav reads consistently on every page:
  ... | Application Calendar | WIP | Commercial Scorecard
Applied to Project Financials, Outstanding Invoices, Retention, Variations,
Application Calendar and Commercial Scorecard. (Contracted Rates and Applications use
the shared CommercialNav, which already draws all separators.)
