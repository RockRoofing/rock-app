Forms — choose Combined vs Separate on multi-download
=====================================================
Deploy target: RockRoofing/rock-app  branch main
Build: ✓ Compiled successfully
Single file: pages/operations/forms.js

CHANGE
------
When 2+ forms are selected in Ops > Forms > Completed and you click Download, a small
dialog now asks how you want them:
  • Separate files — one PDF per form, each downloaded individually (named by
    project + form title).
  • Combined file — all forms in a single PDF (the previous behaviour).
Selecting a single form downloads it directly (no prompt).

Separate downloads are staggered slightly so browsers don't block the multiple
downloads; you may get a one-time "allow multiple downloads" prompt from the browser
the first time.
