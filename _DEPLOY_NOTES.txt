Deploy target: RockRoofing/rock-app  branch main
Build: ✓ Compiled successfully

CHANGE
------
The "⬆ Upload Bills" button on the Bookkeeping page used to navigate away to the
Xero Upload tool. It now opens a MODAL on the same page:
 • Select one OR several Bills CSV exports at once (multi-file).
 • Each file is posted to /api/import-bills-bulk (exact same logic as the Xero
   Upload tool — per-day replace, tagged lines -> projects, untagged -> Overheads).
 • Per-file progress + result shown in the modal.
 • On "Done", the page's figures refresh automatically (re-fetches /api/bookkeeping)
   so new bills/overheads appear without a manual reload.

No API changes — reuses the existing, working import endpoint. The full Xero
Upload tool (wages, invoices, etc.) is still available via Bookkeeping Tools.
