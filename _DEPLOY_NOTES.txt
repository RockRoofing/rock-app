PLANNING — Edit Labour Allocation

  pages/operations/project-management/planning.js
   - New "Edit labour allocation" button in the selection action bar, positioned between
     "Allocate labour →" and "Clear selection". It only appears when the selected cell(s) already
     have labour allocated.
   - Opens the same pop-up in EDIT mode: pre-loaded with everyone currently allocated across the
     selected days (the union). You can remove people or add others. Saving sets that exact list on
     every selected day (so removing someone takes them off those days; an empty list clears them).
   - "Allocate labour" (add mode) is unchanged: it merges the chosen installers onto the days.
   - Clash checking still applies in both modes.

Deploy over the previous planning.js. No API change.
