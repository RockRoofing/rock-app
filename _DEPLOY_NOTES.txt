1) XERO UPLOAD MOVED INTO ADMIN
-------------------------------
• New page: Admin → "Xero Upload" (/admin/xero-upload), added as the LAST tab in
  the Admin sub-nav. Admin-only, no separate back link (uses the Admin nav).
• The "Upload" link has been REMOVED from the Project Financials (commercial) nav.

2) THREE ALL-PROJECTS BULK UPLOADS (one file each, matched by tracking category)
-------------------------------------------------------------------------------
• Bills (Costs) — NEW bulk importer: materials + subcontractor labour for every
  project from the Xero Bills CSV. (Runtime-tested on your file.)
• Direct Wages — NEW bulk importer: PAYE/direct wages (account 320) from the Xero
  Account Transactions CSV, matched per line by the Projects tracking category.
  NOTE: built to the common Account Transactions column layout with flexible
  column detection. If your wages export columns differ, it returns a clear
  message asking to map them — send James a sample wages CSV and it'll be tuned.
• Sales Invoices — the existing bulk importer, now on this page.

3) COSTS NO LONGER OVERWRITE EACH OTHER
---------------------------------------
Bills, Wages and the legacy per-project upload now write to SEPARATE keys
(costs:bills / costs:wages / costs:manual) and are SUMMED into the figure the
dashboard reads (lib/mergeCosts.js). So uploading Bills won't wipe Wages, and
vice-versa — they add together per project.

4) NAV FIX
----------
The commercial-family nav bars (Retention, Outstanding Invoices, Variations,
Application Calendar) could clip/hide links on narrower screens. The nav rows now
scroll horizontally so ALL headers are reachable regardless of which page you're on.
