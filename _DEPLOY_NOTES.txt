NEW: "Upload all files at once"
------------------------------
A single drop-zone at the top of Admin -> Xero Upload accepts MULTIPLE files.
Each file is auto-detected and routed to the right importer:
  • .xlsx / .xls                      -> Direct Wages
  • .csv with Type = "Sales …"        -> Sales Invoices
  • .csv with Type = "Bill …"         -> Bills (Costs)
Detection reads the "Type" column values (not just the filename), so a Bills file
that happens to contain a "Bill credit note" row is still detected as Bills.

Files process one after another with a per-file status line showing what it was
detected as and a result summary (e.g. "Bills (Costs) · 120 new, 15 untagged").
The individual single-type upload areas remain below for one-at-a-time uploads.

NOTE
----
This is purely a front-end convenience — it calls the SAME importers as the
single areas, so all the merge/dedupe/untagged behaviour is identical.
