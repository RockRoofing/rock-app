WHAT CHANGED vs the previous bundle
-----------------------------------
The sync now brings in ONLY accounts that can be a project cost or overhead:
  • Class = EXPENSE (Xero's cost-of-sale + overhead + expense accounts), i.e.
    types DIRECTCOSTS, OVERHEADS, EXPENSE.
  • BALANCE-SHEET accounts (ASSET, LIABILITY, EQUITY) and REVENUE/Sales accounts
    are EXCLUDED — they never appear as a project cost, so they won't clutter the
    categorisation list or the uncategorised banner.

Everything else from the previous bundle is included:
  1) "↻ Sync chart of accounts" button (pulls code + name from Xero).
  2) "Ignore" renamed to "Overheads" (legacy ignore auto-migrated).
  3) New/unassigned codes = "Uncategorised": excluded from project costs + flagged.
  4) Bookkeeping app banner listing uncategorised codes with a link to fix them.
